## Approche pédagogique

Chaque parcours est conçu selon un principe simple : **construire, pas seulement animer.**
Cela signifie :
- Objectifs d'apprentissage clairs, alignés sur le contexte professionnel réel de l'apprenant
- Ancrage dans des théories reconnues (SLA) plutôt que des intuitions
- Séparation nette entre contenu apprenant et notes pédagogiques
- Mesure d'impact systématique (satisfaction, engagement, résultats)
